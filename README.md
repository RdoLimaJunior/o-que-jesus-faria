# ✦ O Que Jesus Faria? — Sabedoria & Salmos ✦

Uma ferramenta espiritual moderna projetada para trazer conforto, sabedoria e reflexão diária através dos ensinamentos de Jesus Cristo e das Escrituras.

## ✨ Funcionalidades

- **🙏 Busca de Sabedoria:** Conte sua situação e receba um conselho baseado nos princípios de Cristo, acompanhado de uma referência bíblica.
- **📖 Meditação em Salmos:** Seleção dos Salmos mais amados com o método *Ler · Pensar · Orar · Viver*.
- **☀️ Devocional Diário:** Um versículo e uma reflexão nova todos os dias para começar sua manhã com fé.
- **📔 Diário de Oração:** Registre suas orações e acompanhe suas conversas com Deus.
- **🎙️ Voz Divina:** Ouça os conselhos e escrituras em voz alta (Integração com ElevenLabs).
- **📱 PWA (Progressive Web App):** Instale o aplicativo no seu celular e use como um app nativo, com suporte offline.

## 🎨 Design Premium

O projeto conta com uma interface **Celestial Design**:
- Fundos dinâmicos com raios de sol e nuvens.
- Tipografia clássica (Cinzel e EB Garamond).
- Micro-animações suaves para uma experiência tranquila.
- Layout totalmente responsivo para celulares e tablets.

## 🛠️ Tecnologias Utilizadas

- **HTML5** (Semântico)
- **CSS3** (Variáveis, Flexbox, Grid e Animações)
- **JavaScript** (Vanilla JS - Puro)
- **ElevenLabs API** (Para narração de áudio premium)
- **Service Workers** (Para suporte Offline/PWA)

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/RdoLimaJunior/o-que-jesus-faria.git
   ```
2. Abra o arquivo `index.html` no seu navegador.
3. (Opcional) Configure sua API Key da ElevenLabs no arquivo `app.js` para habilitar a voz.
